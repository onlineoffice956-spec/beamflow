## ✨ Features

- 📐 **Longitudinal reinforcement** - Calculate tension/compression steel
- 🔄 **Shear stirrups** - Automatic spacing and sizing
- 📏 **Deflection checks** - Span/depth ratio verification
- 🇪🇺 **Eurocode 2 (EN 1992-1-1:2004)** compliant
- 🇬🇧 **BS 8110-1:1997** compliant
- 🚀 **Auto-deploy** - Push to GitHub, automatically deploys
- 🔍 **PR Previews** - Test changes in isolation
- 📊 **Real-time results** - Instant calculations with warnings

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/beamflow.git
cd beamflow

# Install backend dependencies
cd backend
npm install
npm start

# In a new terminal, install frontend
cd ../frontend
npm install
npm start

# Open http://localhost:3000
